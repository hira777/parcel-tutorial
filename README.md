## 使い方

### Parcelのインストール

```bash
npm install -g parcel-bundler
```

or

```bash
yarn global add parcel-bundler
```

### clone

```bash
git clone git@github.com:hira777/parcel-tutorial.git
cd parcel-tutorial
```

### Parcelの実行

``` bash
yarn
yarn run build
```

or

``` bash
npm install
yarn run build
```
